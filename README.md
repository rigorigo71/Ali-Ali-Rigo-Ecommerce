## Product Description

SOLE is an online shoe catalog where customers can browse and discover footwear from top brands. We offer a curated selection of sneakers and classics from Nike, Converse, Adidas, and more.

## Data Model

Each shoe object has the following fields:

| Field | Type | Description |
|-------|------|-------------|
| `name` | `string` | URL-friendly identifier (e.g. `air-max`) |
| `brand` | `string` | Manufacturer name (e.g. `Nike`) |
| `size` | `number` | US shoe size (e.g. `10`) |
